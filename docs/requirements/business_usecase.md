## BUC01: 花束を注文する

| 項目 | 内容 |
| :--- | :--- |
| スコープ | フレール・メモワール WEB ショップ |
| レベル | ユーザー目的 |
| 主アクター | 得意先 |
| 概要 | 得意先が WEB ショップから花束を選び、届け日・届け先・メッセージを指定して注文する |
| 頻度 | 高（日常的） |

## BUC02: 届け日を変更する
