## 対応方針

### 開発フェーズ（IT1-IT2）で対応

- salt 図不足分は実装時にデザインを確定（C-04, C-07, C-10, A-03, A-13）
- 空状態・ローディング・ステップインジケーターは実装時に組み込み
- モバイルレイアウトは Tailwind CSS レスポンシブで実装時に対応

### 設計ドキュメント更新で対応

- ログイン不要ブラウジングの画面遷移図修正
- ロール別ナビゲーションの可視性マトリクス追加

### 許容（現時点では対応不要）

- ダークモード、デザイントークン、アニメーション（実装時に段階的に対応）
