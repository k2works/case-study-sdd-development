# UI 設計 - フレール・メモワール WEB ショップ

## 画面一覧

| # | 画面名 | アクター | 対応 UC | 種類 |
| :--- | :--- | :--- | :--- | :--- |
| P01 | 商品一覧 | 得意先 | UC01 | コレクション |
| P02 | 注文画面 | 得意先 | UC01, UC02 | シングル（作成） |
| P03 | 注文確認 | 得意先 | UC01 | シングル（確認） |
| P04 | 注文完了 | 得意先 | UC01 | シングル（結果） |
| P05 | 受注一覧 | スタッフ | UC04, UC03 | コレクション |
| P06 | 受注詳細 | スタッフ | UC04 | シングル |
| P07 | 在庫推移 | スタッフ | UC05 | コレクション |
| P08 | 発注画面 | スタッフ | UC06 | シングル（作成） |
| P09 | 入荷登録 | スタッフ | UC07 | シングル（作成） |
| P10 | 出荷一覧 | スタッフ | UC08, UC09 | コレクション |
| P11 | 商品管理 | スタッフ | UC10 | コレクション + シングル |
| P12 | 単品管理 | スタッフ | UC11 | コレクション + シングル |

## 画面遷移図

### 得意先向け

```plantuml
@startuml
title 得意先向け画面遷移

[*] --> P01_商品一覧

state P01_商品一覧 : 花束の一覧表示
P01_商品一覧 --> P02_注文画面 : 花束を選択

state P02_注文画面 : 届け日・届け先・メッセージ入力
P02_注文画面 --> P03_注文確認 : 確認ボタン
P02_注文画面 --> P01_商品一覧 : 戻る

state P03_注文確認 : 注文内容の確認
P03_注文確認 --> P04_注文完了 : 注文確定
P03_注文確認 --> P02_注文画面 : 修正

state P04_注文完了 : 注文完了
P04_注文完了 --> P01_商品一覧 : 続けて注文
P04_注文完了 --> [*]

@enduml
```

### スタッフ向け

```plantuml
@startuml
title スタッフ向け管理画面遷移

[*] --> P05_受注一覧

state P05_受注一覧 : 受注状態別一覧
P05_受注一覧 --> P06_受注詳細 : 受注選択

state P06_受注詳細 : 受注情報・届け日変更
P06_受注詳細 --> P05_受注一覧 : 戻る

state P07_在庫推移 : 日別在庫予定数
P07_在庫推移 --> P08_発注画面 : 発注ボタン

state P08_発注画面 : 単品の発注
P08_発注画面 --> P07_在庫推移 : 発注完了/戻る

state P09_入荷登録 : 入荷実績の記録
P09_入荷登録 --> P07_在庫推移 : 登録完了

state P10_出荷一覧 : 出荷日の受注一覧
P10_出荷一覧 --> P10_出荷一覧 : 出荷記録

state P11_商品管理 : 花束の構成管理
state P12_単品管理 : 単品のマスタ管理

@enduml
```

## 画面イメージ

### P01: 商品一覧（得意先向け）

```plantuml
@startsalt
{+
  {/ <b>フレール・メモワール WEB ショップ }
  ---
  <b>花束一覧
  {
    {SI
      {
        <b>ローズブーケ
        赤いバラを中心とした華やかな花束
        <b>¥5,500（税込）
        [ 注文する ]
      }
      ---
      {
        <b>スプリングミックス
        春の花を集めた明るい花束
        <b>¥3,800（税込）
        [ 注文する ]
      }
      ---
      {
        <b>ホワイトリリー
        白いユリの上品な花束
        <b>¥4,200（税込）
        [ 注文する ]
      }
    }
  }
}
@endsalt
```
