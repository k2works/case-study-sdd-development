## 選定基準

| 基準 | 重み | 説明 |
| :--- | :--- | :--- |
| チーム習熟度 | 高 | 1-2 名チームで学習コストを最小化 |
| LTS / 安定性 | 高 | 長期サポートのあるバージョンを優先 |
| コミュニティ | 中 | 問題解決時の情報源の豊富さ |
| Django 親和性 | 高 | Django エコシステム内の技術を優先 |
| 運用コスト | 中 | マネージドサービスを活用して運用負荷を最小化 |
