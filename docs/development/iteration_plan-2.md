### ストーリー詳細

#### US-004: 商品を選択する

**ストーリー**:

> 得意先として、WEB ショップで商品を選び詳細を確認したい。なぜなら、贈る花束を比較検討して選びたいからだ。

**受入条件**:

- [ ] 商品一覧画面で花束のカード形式表示が動作する
- [ ] 商品詳細画面で構成花材・価格・説明が確認できる
- [ ] ログイン不要で商品閲覧が可能

#### US-005: 届け日・届け先・メッセージを入力して注文する

**ストーリー**:

> 得意先として、届け日・届け先・メッセージを入力して注文を確定したい。なぜなら、大切な記念日に花束を届けたいからだ。

**受入条件**:

- [ ] 届け日をカレンダーから選択できる
- [ ] 届け先（氏名・住所・電話番号）を入力できる
- [ ] メッセージカード内容を入力できる（200 文字以内）
- [ ] 注文確認画面で内容を確認できる
- [ ] 注文確定後に注文番号が表示される
